ZSH + TMUX + (N)VIM All the best IDE

Published 3 months ago5 min read4 comments

AS I mentioned in my previous post, I am a big fan of ZSH and TMUX. I use them in my daily work. I have been using them for a long time. also I am a big fan of (N)VIM. I use it for editing my code. at the first i was using it with the default terminal. but after a while, I realized that it is not a good idea. I need to use a better IDE for editing my code. I tried many IDEs. but I did not like them. I realized that I need to use a terminal based IDE. I started to use (N)VIM with ZSH and TMUX. I realized that it is the best IDE for me. Also I am a big fan of Python. I use to work with vim at the first. but after a while, I realized that there is newer version of vim named neovim with more capabilities. I started to use neovim.

Magic of Plugins

For all these tools there is a plugin Manager. and lots of plugins for them. I am using oh_my_zsh for ZSH. and TPM for TMUX. and vim-plug for (N)VIM. I am using these plugins for my daily work. I am going to explain some of them. you can Customize your ZSH with oh_my_zsh. you can install plugins for that also the same for tmux. and for (N)VIM.

Usually,there is config file for each of them. you can customize them throw that file. after configuration you will see the magic of these tools.

the config file for ZSH is .zshrc. the config file for TMUX is .tmux.conf. the config file for (N)VIM is init.vim. you can find these files in your home directory. you can customize them with your favorite text editor. I am using (N)VIM for editing these files.
